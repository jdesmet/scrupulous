Scrupulous.js

Scrupulous.js is a jQuery plugin that provides simple, client-side, inline form validation. It uses HTML5 attributes to control validation along with some custom validation methods. Oh, and it plays nicely with Bootstrap. Check out the demo below, then read the documentation here.
